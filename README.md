# Naver Blog Web Writer

브라우저에서 바로 쓰는 네이버 블로그 작성기입니다.

사진과 경험 메모를 넣으면 네이버 블로그용 글, 썸네일, 태그, 예상 유입 키워드, 예상 광고, 업로드 체크리스트를 한 화면에서 만듭니다. 생성된 글은 작성기 안에서 바로 수정할 수 있습니다.

## 들어간 기능

- 사진 업로드와 미리보기
- 사진별 역할 자동 추천
- 경험 메모 기반 네이버 블로그 원고 생성
- 작성기 안에서 원고 직접 수정
- 제목 A/B 후보
- 썸네일 자동 생성
- 태그 다이어트
- 글자수와 문단 밀도 점검
- 예상 유입 키워드
- 예상 광고 카테고리
- AI 느낌 표현 검사와 빠른 수정
- 말투 프리셋 저장
- 네이버 버전과 블로그스팟/구글 버전 분리 생성
- 업로드 전 체크리스트
- 원고, 리포트, 썸네일 다운로드

## 기본 글 방향

- 실제 경험 중심
- 1인칭 구어체
- 사진 흐름 중심
- SEO는 제목, 소제목, 태그, FAQ에만 자연스럽게 반영
- AI가 쓴 느낌보다 내가 다녀온 기록 느낌
